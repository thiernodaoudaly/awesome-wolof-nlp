# Contributing to awesome-wolof-nlp

Thank you for helping grow this resource! Every addition — a dataset, a paper, a tool — makes a real difference for the Wolof NLP community.

---

## What to contribute

- **Datasets** — corpora, lexicons, annotated data, audio resources
- **Models** — pre-trained models, embeddings, fine-tuned checkpoints
- **Tools** — tokenizers, taggers, evaluation scripts, preprocessing utilities
- **Papers** — peer-reviewed research, workshop papers, theses
- **Groups & Labs** — research teams, community initiatives, university labs
- **Related languages** — resources for Pulaar, Serer, Mandinka, or other West African languages useful for cross-lingual work

If you're unsure whether something belongs here, open an issue and ask. When in doubt, include it.

---

## Before you submit

1. **Check for duplicates** — search the README to make sure the resource isn't already listed.
2. **Verify the link works** — broken links will be rejected.
3. **Make sure the resource is publicly accessible** — no paywalls or login-required pages.
4. **Write a comment** — every entry must include 1–2 sentences explaining what the resource is and why it's useful. Entries without comments will not be merged.

---

## Format

Follow the existing format exactly:

```markdown
- **[Resource Name](https://link.to/resource)** — What it is, what it contains, and when to use it. *Optional note on quality, limitations, or comparison to similar resources.*
```

For papers, always include the year:

```markdown
- **[Paper Title (YEAR)](https://arxiv.org/abs/XXXX)** — What the paper is about. *Who should read it and why.*
```

---

## How to submit a PR

1. Fork this repository.
2. Edit `README.md` — add your entry in the right section, in alphabetical order if possible.
3. Commit with a clear message, e.g. `Add Leipzig Corpora for Wolof`.
4. Open a Pull Request with a short description of what you're adding and why.

---

## What will be rejected

- Resources behind paywalls or requiring registration
- Entries without a comment/description
- Duplicate entries
- Broken or unstable links
- Self-promotion without clear community value
- Resources unrelated to Wolof or West African NLP

---

## Reporting issues

Found a broken link? Outdated resource? Open an issue with the label `broken link` or `outdated` and we'll fix it.

---

Thanks for contributing — every entry helps build the ecosystem.
